\### Reversible wave–particle collapse



We model collapse as local compression of a wavefunction:



&#x20;   psi\_coll(x) = psi(x) \* K(x - x0)



where:

\- psi(x) is the extended wave state

\- psi\_coll(x) is the localized particle-like state

\- K(x - x0) is a Gaussian kernel centered at x0



If K(x - x0) is never zero, we can invert the operation:



&#x20;   psi(x) = psi\_coll(x) / K(x - x0)



So:



&#x20;   psi(x) → psi\_coll(x) = psi(x) \* K(x - x0)

&#x20;   psi\_coll(x) → psi(x) = psi\_coll(x) / K(x - x0)



This shows that, in this model, the particle state still contains all the information of the original wave, and collapse is a reversible compression, not destruction.
