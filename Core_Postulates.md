This document defines the fundamental axioms of the Information‑Compression Model (ICM).

These postulates form the theoretical foundation for all predictions, simulations, and experiments in this repository.



The model treats quantum behavior as transitions between information states, governed by reversible compression and decompression processes.



INFORMATION IS NEVER DESTROYED



Postulate:

All quantum processes preserve total information.

Collapse does not delete information; it compresses it.



Implication:

Particle‑like behavior corresponds to compressed information.

Wave‑like behavior corresponds to decompressed information.



COLLAPSE IS A REVERSIBLE COMPRESSION OPERATION



Postulate:

Measurement‑induced collapse is implemented as a reversible

information‑compression transform.



In plain text:

Collapse = multiply by a compression kernel

Uncollapse = divide by the same kernel



Implication:

Collapse does not eliminate the wave; it compresses it into a

localized packet that still contains all original information.



A CONTINUOUS COMPRESSION PARAMETER c DEFINES THE STATE



Postulate:

Every quantum system has a compression value c between 0 and 1.



c = 0  -> fully wave‑like (decompressed)

c = 1  -> fully particle‑like (compressed)



Implication:

Quantum behavior is not binary.

It is a continuous spectrum controlled by c.



MEASUREMENT FORCES c TOWARD 1



Postulate:

Any measurement interaction increases compression.



Weak measurement -> small increase in c

Strong measurement -> large increase in c

Projective measurement -> c approaches 1



Implication:

Measurement is not a mysterious discontinuity.

It is a controlled information‑compression process.



ERASURE FORCES c TOWARD 0



Postulate:

Removing which‑path information decompresses the state.



Erasure reduces c.

Full erasure returns c to 0.



Implication:

Quantum eraser experiments are decompression operations.



COLLAPSE SHAPE IS DETERMINED BY THE DETECTOR GEOMETRY



Postulate:

The spatial form of the compressed packet depends directly on

the geometry and response profile of the detector.



Implication:

Collapse is not universal.

It is shaped by the measurement apparatus.



COMPRESSION AND DECOMPRESSION ARE HISTORY‑DEPENDENT



Postulate:

The sequence of compression and decompression operations affects

the final state.



Implication:

The model predicts hysteresis:

compress -> erase -> compress

is not equivalent to:

compress (once) to the same final level



MULTI‑PASS COMPRESSION COMPOUNDS NON‑LINEARLY



Postulate:

Sequential compressions do not combine linearly.



Implication:

Multiple weak measurements produce non‑linear visibility curves,

unlike the exponential decay predicted by standard QM.



COLLAPSE STRENGTH IS CONTROLLED BY A KERNEL WIDTH (SIGMA)



Postulate:

The compression kernel has a width parameter sigma.



Large sigma  -> gentle collapse

Small sigma  -> strong collapse



Implication:

Collapse strength is tunable and continuous.



PARTIAL COLLAPSE PRESERVES HIDDEN STRUCTURE



Postulate:

Partial compression retains recoverable sub‑interference patterns.



Implication:

Applying an approximate inverse kernel can reconstruct hidden

structure that standard QM treats as irreversibly lost.



COMPLEMENTARITY IS LINEAR IN THE ICM FRAMEWORK



Postulate:

Distinguishability D and visibility V satisfy:



D = c

V = 1 - c



Implication:

Complementarity is linear, not quadratic.



ALL QUANTUM BEHAVIOR EMERGES FROM INFORMATION FLOW



Postulate:

Wave‑particle duality, interference, collapse, and entanglement

are all manifestations of information compression and

decompression dynamics.



Implication:

The model provides a unified information‑theoretic interpretation

of quantum mechanics.

Information is preserved

Measurement increases c

Detector geometry shapes collapse
